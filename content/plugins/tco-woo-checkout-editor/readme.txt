Manage your WooCommerce checkout fields

== Description ==
Order, create and edit your WooCommerce checkout fields for the checkout page. An easy to use
admin interface that allows you to edit default fields and add/remove your own custom fields.


== Installation ==

1. Make sure you have the latest version of WooCommerce plugin installed
1. Upload the extracted plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to the admin menu WooCommerce -> 'Checkout Manager'
